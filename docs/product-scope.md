# 当前 MVP 范围

本 MVP 只服务单家庭或少量家庭内测，目标是跑通家庭训练最短闭环：

照护者设置训练难度，系统生成今日训练，患者完成数学题和数字顺序练习，系统提示写名字和跟唱，家属查看训练结果并保存每日记录。

## 产品边界

- 仅作为家庭日常认知活动辅助和训练记录工具。
- 不替代医生或专业人员评估。
- 不输出医学判断、疾病分期、治疗效果或药物建议。
- 患者端强调低压力、鼓励式反馈。
- 家属端提供详细记录和备注能力。
- 写名字任务可以使用纸笔，也提供设备触屏书写区；当前不保存笔迹、不识别、不评分。
- 训练过程使用本地草稿保存，完成家属记录后再进入历史记录。
- 训练开始前可记录今日状态，用于产品层面的练习节奏提示。
- 训练结束后提供下次设置建议，建议只围绕题量、难度、任务数量和参与意愿。
- 历史详情默认只读，家属可主动进入编辑模式修改备注和状态。
- 日常训练包含看图说名称、图形临摹和找不同题型；看图说名称的语音输入和找不同记录客观作答，触屏手写/手绘不做 OCR 识别，只保存输入痕迹供家属查看。
